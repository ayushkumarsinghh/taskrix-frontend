# Taskrix Frontend

A premium, queue-driven task management system built with React, TypeScript, and Tailwind CSS.

## Features

- **Atomic Execution**: Manage tasks with precision using a queue-driven approach.
- **Premium UI**: Modern design with glassmorphism, smooth animations (Framer Motion), and a curated color palette.
- **Responsive Dashboard**: Real-time task tracking and management.
- **Secure Auth**: Full authentication flow integrated with a robust backend.

## Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **State Management**: React Hooks
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **HTTP Client**: [Axios](https://axios-http.com/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
